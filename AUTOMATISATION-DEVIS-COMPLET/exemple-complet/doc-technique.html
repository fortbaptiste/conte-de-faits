06 Sécurité & Données

Protection des données
  • HTTPS sur tous les endpoints
  • Authentification API (tokens)
  • Données client jamais envoyées au LLM
  • Logs anonymisés
  • Backup quotidien PostgreSQL
Propriété
  • Code workflows : propriété client
  • Données : 100% propriété client
  • Mémoire produits : propriété client
  • Articles générés : propriété client
